Quick Reference Card

The facts below are most commonly tested. Read through before the mock exam.

Chapter 1 — AI Overview
  • 3 AI schools: Symbolism, Connectionism, Behaviorism
  • 3 elements: Data, Computing Power, Algorithms
  • Transformer types: Encoder-only (BERT), Decoder-only (GPT), Both (T5)
  • PanGu tiers: Base LLM → Industry → Scenario
  • Emergence: unexpected capability from scale alone
  • AI eras: Perception → Understanding → Creation & Generation
Chapter 2 — Machine Learning
  • Feature selection: Filter (statistics), Wrapper (trains model), Embedded (L1/tree)
  • GD variants: BGD (all), SGD (one sample), MBGD (mini-batch)
  • Overfitting → high variance; Underfitting → high bias
  • k-Means, Hierarchical = unsupervised
  • Random Forest = Bagging; GBDT = Boosting
  • Cross-validation: prevents overfitting to one train/val split
Chapter 3 — Deep Learning
  • ReLU: solves vanishing gradient; default activation
  • CNN output: (Input − Kernel + 2P) / Stride + 1
  • Max Pooling: no learnable parameters
  • LSTM gates: Forget, Input, Output + Cell State
  • Attention: softmax(QKᵀ / √dₖ) · V
  • BERT = bidirectional encoder; GPT = left-to-right decoder
  • Adam = Momentum + RMSProp (default optimiser)
  • Dropout: ON during training, OFF during inference
  • MoE: router activates top-k experts per token
Chapter 4 & Advanced
  • MindSpore base class: nn.Cell; forward method: construct()
  • MindSpore modes: Graph (static), PyNative (dynamic)
  • LeNet: Conv→Pool→Conv→Pool→FC→FC→Output (10 classes)
  • 6-stage pipeline: Req → Env → Data → Network → Train → Deploy
  • Federated Learning: only weight updates (not raw data) shared
  • Transfer: small data → feature extraction (freeze backbone)
  • Precision = TP/(TP+FP); Recall = TP/(TP+FN)
  • AUC = 0.5 (random), 1.0 (perfect)

Exam Day Tips

Know the MindSpore vocabulary cold
The exam tests nn.Cell vs nn.Module, construct() vs forward(), Graph Mode vs PyNative, and MindSpore Serving vs Lite. These are easy marks if memorised.
Use the output size formula on any CNN question
Write it out: (Input − Kernel + 2P) / Stride + 1. Plug in the numbers. Don't guess.
LSTM: know all three gate names and what each does
Forget = erase old memory. Input + Candidate = write new memory. Output = expose cell state as hidden state. Any question mentioning "cell state" or "long-range dependency" is about LSTM.
Match metric to scenario instantly
Miss a cancer case = worst → Recall. False alarm is costly → Precision. Imbalanced classes → F1. Comparing models → AUC.
Three AI schools, three elements
Schools: Symbolism / Connectionism / Behaviorism. Elements: Data / Computing Power / Algorithms. These appear in early Chapter 1 questions — don't lose easy marks here.
Flag and skip, don't get stuck
90 minutes for ~60 questions = 90 seconds each. If a question requires long mental arithmetic, flag it, move on, come back. The scoring is equal per question.
