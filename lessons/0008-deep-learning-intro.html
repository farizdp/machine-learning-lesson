Deep Learning: Perceptrons & Neural Networks

How the brain inspired a revolution in AI — and how a chain of simple math operations can learn almost anything.

From ML Algorithms to Deep Learning

In Chapter 02 you learned algorithms like Decision Trees and SVMs — each hand-crafted to solve a specific type of problem. Deep Learning takes a different approach: instead of designing the algorithm, you design the structure of a network, and the network learns everything else from data.

The building block of every neural network — from a simple classifier to GPT-4 — is the perceptron.

1. What Is a Perceptron?

A perceptron is a mathematical model of a single biological neuron. Just as a brain neuron receives signals, combines them, and fires if the combined signal is strong enough — a perceptron receives numbers, multiplies each by a weight, sums them, and produces an output.

Biological Analogy
A real neuron receives electrical signals through dendrites, combines them in the cell body, and fires an output signal through the axon if the total input crosses a threshold. A perceptron does exactly this — in math.

The perceptron: weighted sum of inputs plus bias, then an activation function f.

Inputs (x₁, x₂, x₃…)
The features fed into the neuron — numbers representing your data (e.g. pixel brightness, temperature, word count).
Weights (w₁, w₂, w₃…)
How much each input matters. A large positive weight amplifies the input; a negative weight suppresses it. Weights are learned during training.
Bias (b)
A constant that shifts the output up or down, independent of the inputs. Gives the neuron flexibility to fire even when all inputs are zero. Also learned during training.
Activation function f(·)
Applied to the weighted sum. Without it, the perceptron is just linear arithmetic — and stacking linear operations gives you nothing extra (more on this shortly).

What can a single perceptron do?

A single perceptron can learn a linear decision boundary — it can separate two classes with a straight line. It works for simple problems (AND gate, OR gate) but fails on anything non-linear (XOR gate, complex image patterns).

⚠ Exam Alert
A single perceptron can only solve linearly separable problems. It cannot solve XOR — this limitation was famously identified in 1969 and temporarily halted AI research. The solution: stack perceptrons into multi-layer networks.

2. Nonlinearity — Why It's Essential

Before stacking layers, we need to understand why stacking linear layers alone is mathematically pointless.

If every neuron just computes w·x + b (a linear function), then no matter how many layers you add, the whole network is still one big linear function. You could always collapse 100 linear layers into a single equivalent layer — depth adds zero expressive power.

Without Nonlinearity
Layer 1: y = W₁x + b₁
Layer 2: z = W₂y + b₂
Combined: z = (W₂W₁)x + c

Still just one linear layer. No matter how deep you go, you gain nothing new.
With Nonlinearity
Layer 1: a = f(W₁x + b₁)
Layer 2: z = f(W₂a + b₂)

Because f is nonlinear, these cannot be collapsed. Each layer learns genuinely new representations.

The activation function is what introduces nonlinearity after each layer's weighted sum. It is what makes "deep" in deep learning meaningful — enabling networks to learn edges, curves, shapes, words, and strategies from raw data.

⚠ Exam Alert
Without nonlinear activation functions, stacking more layers gives no extra expressive power. Nonlinearity is the core ingredient that allows deep networks to learn complex patterns.

3. Multi-layer Perceptron (MLP)

Stack multiple layers of neurons together and you get a Multi-layer Perceptron (MLP), also called a Feedforward Neural Network or Fully-connected Neural Network.

Input Layer
Receives the raw data. Each node is one feature. No computation happens here — it just passes values forward.
Hidden Layers
Where the learning happens. Each neuron computes a weighted sum of all its inputs, adds a bias, then applies an activation function. "Hidden" because their values are not directly observed from outside the network.
Output Layer
Produces the final prediction. For classification, each node represents one class — the node with the highest score wins.
Fully Connected (Dense)
Every neuron in one layer connects to every neuron in the next. All weights are learned — this is why a large network has millions of parameters.

4. Deep Neural Network (DNN)

A Deep Neural Network is an MLP with many hidden layers — typically 3 or more. The word "deep" refers to the number of layers (the network's depth).

Analogy — Feature Hierarchy in Image Recognition
A deep network recognizing a face learns a hierarchy without being told:
  • Layer 1 detects edges (horizontal, vertical, diagonal lines)
  • Layer 2 combines edges into shapes (circles, curves, corners)
  • Layer 3 combines shapes into parts (eyes, nose, mouth)
  • Layer 4+ combines parts into a full face → identifies the person
Each layer builds on the previous one. You never programmed any of this — the network discovered it from data alone.

The Forward Pass

Data flows left to right from input to output. This single forward journey through the network is called the forward pass.

Input x
raw features
→
Layer 1
z = Wx + b
a = f(z)
→
Layer 2
z = Wa + b
a = f(z)
→ … →
Output ŷ
prediction

At each layer: z = Wx + b (weighted sum), then a = f(z) (activation). Pass a to the next layer.

Pre-activation (z)
z = W·a_prev + b — the raw weighted sum before the activation function. Also called the "logit."
Post-activation (a)
a = f(z) — the output after applying the activation function. This becomes the input to the next layer.

5. How Depth Increases Capability

In theory, a network with one sufficiently wide hidden layer can approximate any function (Universal Approximation Theorem). In practice, deep networks are far more efficient than wide shallow ones:

  • They build hierarchical representations (pixels → edges → objects)
  • Each layer reuses features built by previous layers
  • They need far fewer total neurons to achieve the same capability
Architecture Capability Typical use
Single perceptron Linear boundary only Linearly separable problems
1 hidden layer (MLP) Any smooth curve (in theory) Basic pattern recognition
2–5 hidden layers Hierarchical features Images, audio, text
Many layers (10s–100s) Very abstract representations GPT, image recognition, AlphaFold
⚠ Exam Alert
Know the three layer types: Input (no computation), Hidden (learns features), Output (prediction). The forward pass computes z = Wx + b then a = f(z) at each layer. "Deep" means many hidden layers. Nonlinearity is what makes depth valuable.
