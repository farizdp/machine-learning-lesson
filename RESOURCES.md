# Machine Learning & AI Resources

## Knowledge

- [Book: _Hands-On Machine Learning with Scikit-Learn, Keras, and TensorFlow_ — Aurélien Géron (O'Reilly)](https://www.oreilly.com/library/view/hands-on-machine-learning/9781098125967/)
  Best beginner-to-intermediate ML book. Use for: supervised/unsupervised ML chapters, neural network foundations.

- [Course: _Machine Learning Specialization_ — Andrew Ng (Coursera/DeepLearning.AI)](https://www.coursera.org/specializations/machine-learning-introduction)
  Gold-standard free course. Use for: linear regression, logistic regression, neural networks, decision trees. Watch lectures alongside lessons.

- [Course: _Deep Learning Specialization_ — Andrew Ng (Coursera/DeepLearning.AI)](https://www.coursera.org/specializations/deep-learning)
  Best resource for CNNs, RNNs, optimization, and sequence models. Use for: Lessons 3+ (deep learning content).

- [Article: _The Illustrated Transformer_ — Jay Alammar](https://jalammar.github.io/illustrated-transformer/)
  Best visual explanation of the Transformer architecture. Use for: Lesson on Attention & Foundation Models.

- [Article: _A Visual Introduction to Machine Learning_ — R2D3](http://www.r2d3.us/visual-intro-to-machine-learning-part-1/)
  Animated visual of how decision trees work. Use for: first exposure to what ML actually does.

- [Docs: _PyTorch Official Tutorials_](https://pytorch.org/tutorials/)
  Use for: AI Development Framework lessons (Chapter 04).

- [Docs: _MindSpore Official Documentation_](https://www.mindspore.cn/en)
  Use for: Huawei-specific framework content in exam syllabus.

- [Paper: _Attention Is All You Need_ — Vaswani et al. (2017)](https://arxiv.org/abs/1706.03762)
  Original Transformer paper. Reference for deep understanding; not required for the exam.
