# Mission: Machine Learning & AI (Certification)

## Why
Pass a professional AI/ML certification exam (syllabus matches Huawei HCIA-AI). Every lesson is built around exam-relevant understanding — not just surface exposure, but genuine comprehension that holds up under exam pressure.

## Success looks like
- Able to explain core AI concepts clearly: intelligence, ML types, neural networks, transformers
- Able to describe the machine learning process end-to-end (data prep → deployment)
- Can distinguish between key algorithms (SVM, decision trees, k-means, etc.) and know when each is used
- Understands deep learning building blocks: perceptrons, CNNs, RNNs, attention, transformers
- Familiar with AI frameworks (PyTorch, MindSpore) and the AI development workflow
- Comfortable with DeepSeek, Foundation Models, GPT, BERT concepts for exam questions

## Constraints
- Beginner level: basic Python but no ML/AI experience
- Learn through structured lessons following the exam syllabus outline
- Needs clear explanations without unnecessary jargon

## Out of scope
- Deep coding implementations (focus is conceptual understanding for exam)
- Advanced research-level topics beyond the syllabus
- Non-Python frameworks (for now)
